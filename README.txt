Wifirst_keepalive
=================

Purpose of this program: Keep alive a Wifirst connection

Use :
- Copy/paste script wifirst_keepalive.tcl on your local host
- Replace "YOUR_LOGIN" and "YOUR_PASSWORD" fields by your Wifirst identifiants
- Under Linux, run "tclsh wifirst_keepalive.tcl&"
- Let script run as long as you need functional connection
