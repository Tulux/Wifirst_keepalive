Wifirst_keepalive
=================

Objectif de ce programme : Automatiser la procédure de connexion au système Wifirst.

Prérequis :
- Système Unix ou dérivés (Linux, *BSD, etc.). Fonctionne sous Windows probable avec l'environnement Cygwin.
- Package tclsh
- Package cUrl

Utilisation :
- Copiez/collez le script wifirst_keepalive.tcl sur votre machine locale
- Remplacez YOUR_LOGIN et YOUR_PASSWORD par vos identifiants Wifirst
- Sous Linux, lancez la commande "tclsh wifirst_keepalive.tcl&"
- Laissez tourner le script aussi longtemps que vous désirez maintenir la connexion
